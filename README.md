# 🦈 Shark Tank India - Startup Investment Dashboard

A Power BI project analyzing startup investments made on the reality show **Shark Tank India**. This dashboard provides insights into investment trends, top-performing startups, shark-wise contributions, and sector-wise funding patterns.

---

## 📊 Project Objective

To visually analyze and interpret investment patterns and co-investment behavior among the sharks featured in Shark Tank India. The goal is to provide data-driven insights on:

- Total investments per season
- Popular startup sectors
- Individual and collaborative shark investments
- Funding trends across founders and pitches

---

## ⚙️ Key Features of Dashboard

- 💰 **Total Investment Summary**  
  View total money invested, number of deals, and average investment.

- 👥 **Shark-wise Contribution**  
  Breakdown of individual and co-investments made by each shark.

- 🧠 **Sector Analysis**  
  Discover which industries attracted the most funding.

- 🔗 **Co-Investment Matrix**  
  Explore partnerships between sharks across deals.

- 📈 **Trends Over Time**  
  Visualize season-wise funding growth and founder pitch success.

---

## 🧹 Data Cleaning Steps

Performed in Python before importing into Power BI:
- Removed nulls and duplicate rows
- Standardized column names and formats
- Unpivoted individual shark investments into a long format
- Replaced missing investment amounts with zeros where needed
- Parsed dates and cleaned sector names
